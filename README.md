# Student-Grade-Checker

A beginner-friendly DSA project developed using C++.

## Features
- Add Student
- Display Students
- Search Student
- Delete Student
- Automatic Grade Calculation

## Technologies Used
- C++
- unordered_map (Hash Table)
- Structures
- Functions

## DSA Concepts Used
- Hash Table
- Searching
- Deletion
- CRUD Operations

## Grade Logic

| Marks | Grade |
|-------|-------|
| 90+ | A |
| 80-89 | B |
| 70-79 | C |
| 60-69 | D |
| Below 60 | F |
